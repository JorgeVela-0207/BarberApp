# BarberApp — Guía rápida para el cliente

## 1. Primera instalación

1. Abre **BarberApp**.
2. Copia el **Device ID** que aparece en pantalla.
3. Envía ese ID a soporte; recibirás un **token** de licencia.
4. Pega el token y pulsa **Activar licencia** (necesitas internet).
5. Crea la cuenta **administrador** (nombre del negocio, usuario y contraseña).
6. Inicia sesión.

> **¿Cambiaste de computadora?** Copia el nuevo Device ID y contacta soporte con tu token. Ellos migrarán la licencia al nuevo equipo.

---

## 2. Primer respaldo (importante)

1. Entra como **administrador**.
2. Ve a **Ajustes** (⚙ en el Dashboard).
3. Pulsa **Exportar respaldo JSON** y guarda el archivo en USB, nube o correo.
4. Repite **al menos una vez por semana**.

La app también guarda respaldos automáticos en el equipo, pero el export manual te protege si falla el disco.

---

## 3. Flujo del día a día

| Momento | Qué hacer |
|---------|-----------|
| **Inicio del día** | Login → revisar Dashboard (citas pendientes, ingresos). |
| **Agendar** | **Agenda** → nueva cita (cliente, servicio, barbero, hora). |
| **Durante el día** | Marcar citas completadas; usar WhatsApp para recordatorios si aplica. |
| **Cobrar** | **Caja** → cobrar citas pendientes (efectivo, tarjeta o transferencia). |
| **Sin cita previa** | **Caja → + Cobro walk-in** (cliente que llegó sin agendar). |
| **Catálogos** | Dar de alta clientes, servicios y personal cuando haga falta. |
| **Cierre** | Revisar totales en Caja; exportar reporte si lo necesitas. |
| **Salir** | **Salir** en Dashboard (cierra sesión). |

---

## 4. Roles

- **Administrador**: acceso total, Ajustes, catálogos completos, reportes.
- **Barbero / estilista**: sus citas, clientes y cobros asignados.

---

## 5. Licencia

- Si la licencia vence, la app mostrará aviso unos días antes.
- Al vencer, aparecerá pantalla de suspensión → **Verificar pago / renovación** (con internet) o contacta soporte.
